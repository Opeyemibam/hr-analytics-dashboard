# HR Analytics Dashboard

## Objective
Analyze employee attrition trends and workforce metrics.

## Tools Used
- Power BI
- Excel

## Key Insights
- High attrition among overtime employees
- Sales department had highest turnover
- Younger employees showed higher churn
